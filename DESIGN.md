2026 Bingo Card - UI Description
Overall Layout

Portrait-oriented card with soft cream/off-white background
Thin gold/tan border framing the entire card
Centered content with generous padding

Header

Year "2026" in light tan/taupe color, smaller serif font
"BINGO" in large, elegant serif font (dark brown/charcoal), all caps, below the year

Grid Structure

5x5 grid of squares
Alternating color pattern between:

Light beige/cream squares (lighter)
Warm taupe/tan squares (darker, muted brown)

The pattern creates a subtle checkerboard effect but not strictly alternating
Rounded corners on each square
Small gaps/gutters between squares

Center Square (Free Space)

Contains a heart icon in dark taupe/brown
Same tan background as other "dark" squares

Square Content

Text is centered within each square
Dark brown/charcoal text color
Small, readable sans-serif or clean serif font
Multi-line text wraps within squares

Inspirational quote in tan/taupe italic serif font
Centered below the grid. This is where the quote/intention will go.

Color Palette

Background: #F9F6F1 (warm off-white)
Light squares: #EDE6DD (light beige)
Dark squares: #C4B5A5 (warm taupe/tan)
Text: #4A4A4A (dark charcoal)
Accent text (header/footer): #B8A898 (muted tan)

Typography

Headers: Elegant serif (like Playfair Display or similar)
Grid text: Clean, readable font, I'd like to use monospace
Footer: Italic serif
